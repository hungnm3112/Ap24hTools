# Lộ Trình Học Tập (Lesson Progress)

## 🎯 Các Kỹ Năng Đã Thông Thạo (Mastered Skills)
1. **Kiến trúc Next.js App Router**: Phân biệt được sự khác nhau giữa Client Component (`'use client'`) và Server Component (mặc định).
2. **Next.js Server Actions**: Biết cách tạo và gọi các Server Actions (`'use server'`) như những API thu nhỏ giấu kín logic ở Backend.
3. **Quản lý Trạng Thái (State) trong React**: Hiểu và vận dụng thành thạo hook `useState` để điều khiển UI (ví dụ: trạng thái `loading`, chuyển đổi UI).
4. **Sử dụng Hook Điều Hướng (Routing)**: Dùng `useRouter` từ `next/navigation` để chuyển trang lập trình (`router.push`).
5. **Giao Diện Ant Design (Cơ bản)**: Biết cách sử dụng form validation, binding dữ liệu `<Form>`, `<Input>`, `<Button>`, `message`.
6. **Thiết lập Mock Data**: Khai báo và sử dụng biến môi trường `.env` kết hợp Promise delay để giả lập mạng.

## 🚀 Các Kỹ Năng Mới Sẽ Học Ở Phase Tiếp Theo (Phase 2 & 3)
1. **Layout Admin (Nested Routes)**: Xây dựng Dashboard (Sidebar, Header) bằng sức mạnh Nested Layout của Next.js.
2. **Quản lý State Toàn Cục (Global State)**: Lưu trữ thông tin User sau đăng nhập (ví dụ dùng Zustand hoặc Context API).
3. **Phân quyền bằng Middleware**: Chặn người dùng chưa đăng nhập truy cập vào `/dashboard`.
4. **Data Fetching nâng cao**: Gọi và hiển thị dữ liệu bảng (Table) từ API thật.
5. **UI Component phức tạp**: Sử dụng Modal, Drawer, Table Pagination của Ant Design.
